Comandos Descriccion
git status Nos da el estado actual de nuestros archivos del repositorio
git add . Prepara los archivos (etapa staged) para ser commitiados (confirmar los cambios)
git commit -m "mensaje de commit" Confirma los cambios que estaban en preparacion (es staged)
git commit -am "mensaje de commit" Agrega y confirma los cambios que estaban en preparacion (es staged) en un solo caomando
git reset --soft HEAD~1 Deshace el último commit de mi repo local (no publicado) manteniendo los cambios
git commit --amend -m "Este es el mensaje correcto" Arregla el mensaje que has usado para el último commit
git log Listar los commits que tengo en emi historial
git branch nombreRama Crea una nueva rama con el nombre especifico
git branch Listar las ramas que tengo en mi repo local
git checkout nombreRama Saltar a la rama especifica
git checkout -b nombreRama Crea y salta a una nueva la rama especifica
git merge nombreRama Une dos ramas
git remote -v Me da info de los repositorios remotos vinculados con mi repo local
git push origin nombreRama Sube cambios a mi repo remoto
git pull origin nombreRama Baja la rama remota con sus cambios a mi repo local
